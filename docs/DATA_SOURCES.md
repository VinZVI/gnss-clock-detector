# Источники данных

Проект агрегирует данные из двух основных независимых источников для обеспечения максимального покрытия.

## 1. GLONASS-IAC FTP (Информационно-аналитический центр КВНО)
**URL:** `ftp.glonass-iac.ru`

Основной источник данных по группировкам ГЛОНАСС и GPS.
- **Путь:** `/MCC/PRODUCTS/ultra/` (или `final`/`rapid`). Файлы статусов находятся в `/MCC/STATUS/YYYY/`.
- **Формат имен:** `Stark_YYDDDHH.sp3` (или `.clk`).
- **Особенности:** Использует проприетарные расширения для метаданных (`.glo`, `.hlt`), которые парсятся для получения информации о состоянии спутников.

## 2. NASA CDDIS HTTPS
**URL:** `cddis.nasa.gov`

Мировой архив GNSS-данных. Используется для получения продуктов IGS (International GNSS Service).
- **Продукты:** `igu` (ultra-rapid), `igr` (rapid), `igs` (final).
- **Формат:** Long-name формат IGS3 (например, `IGS0OPSULT_20261160000_02D_30S_CLK.CLK.gz`).
- **Аутентификация:** Требует NASA Earthdata Login. Реализован автоматический обмен Basic Auth на Bearer Token для обхода проблем с TLS-редиректами на Windows.

## Поддерживаемые форматы файлов
- **SP3 (Standard Product 3):** Позиции спутников и поправки часов.
- **RINEX CLK:** Специализированный формат для поправок часов с высоким разрешением.
- **HLT/GLO:** Текстовые файлы статусов и метаданных ГЛОНАСС-ИАЦ.
